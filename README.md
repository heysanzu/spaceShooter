# Space Shooter
A fun Space Shooter game built using Python and Pygame.

# Features
- Smooth player movement
- Auto shooting
- Increasing difficulty
- Background music
- High score system
- Restart option

# Controls
- A / D or Arrow Keys → Move
- Hold SPACE → Shoot
- SPACE (after game over) → Restart

# Requirements
- Python 3.x
- Pygame

pip install pygame
